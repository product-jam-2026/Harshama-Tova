# Harshama Tova

A registration and management system for groups and workshops of the "Adama Tova" community space.

---


## Main Features
- Registration and management of groups and workshops
- One-click registration for users
- Personalized filtering for activities and groups
- Gender-adapted messaging and experience
- Admin dashboard and management
- Push notifications for users
- Excel export
- Full Hebrew (RTL) support
- Supabase integration

---

## Technologies Used
- Next.js (React framework)
- TypeScript
- Supabase (Database & Auth)
- Web Push API
- MUI (Material UI)
- Styled Components
- SWR (data fetching)
- ExcelJS / xlsx (Excel export)
- Vercel (deployment)
- CSS Modules
